CampusSurvival
==============

Created by:
Jon, Josh and Dillon

Play is currently only supported via local multiplayer.


Player 1: Arrows, Space to shoot, Shift for Material use
Player 2: WASD, Left Shift to shoot, Q for Material use

Don't die!
